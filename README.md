# REACTJS PROJECT: **FRONTEND SHOP HAPI HAPI**

**Hapi hapi** — інтерактивний веб-додаток інтернет-магазин з можливістю оформлення замовлень, управління товарами та користувацькими профілями.

### Технології проекту:
- **JavaScriptXML (JSX)** — Розширення JavaScript дозволяє використовувати HTML разом з JavaScript.
- **React** — бібліотека для інтерфейсу користувача (UI) в JavaScript.

<details>
    <summary>Натисніть, щоб побачити опис технологій цього проекту</summary>
    
    1. JavaScriptXML (JSX) - розширення, яке дозволяє працювати з HTML розміткою для створення структури сторінки веб-додатку.

    2. React - бібліотека JavaScript для створення інтерактивних інтерфейсів (UI) за допомогою компонентного підходу, що дозволяє повторно використовувати код, керувати станом програми і малювати динамічні дані без перезавантаження сторінки.


</details>

### Структура проекту
<details>
    <summary>Натисніть, щоб побачити пояснення до структури цього проекту</summary>
    public - Папка, в якій знаходяться статичні файли
    
    src - Папка, в якій знаходяться уся структура веб додатку.

    <summary>Натисніть, щоб побачити пояснення кожної окремої папки у папці src</summary>

### Стиль написання коду

Код у проєкті написано за допомогою **JavaScriptXML**, що дозволяє працювати з HTML та JavaScript водночас.

У цьому проекті є логічні правила, за допомогою яких відбувається структуризація та **неймінг** файлів. Наприклад:

1. Усі файли називаються через точку, приклад: name.module.css

2. Усі компоненти називаються з великої літери, приклад: Name.tsx

3. Усі функції і константи називаються за принципом **camelCase**, приклад: nameName(){}
