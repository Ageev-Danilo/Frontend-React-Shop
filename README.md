# REACTJS PROJECT: **FRONTEND SHOP HAPI HAPI**

**Hapi hapi** — інтерактивний веб-додаток інтернет-магазин з можливістю оформлення замовлень, управління товарами та користувацькими профілями.

### Технології проекту:
- **JavaScriptXML (JSX)** — Розширення JavaScript дозволяє використовувати HTML разом з JavaScript.
- **React** — бібліотека для інтерфейсу користувача (UI) в JavaScript.

<details>
    <summary>Натисніть, щоб побачити опис технологій цього проекту</summary>
    
    1. JavaScriptXML (JSX) - розширення, яке дозволяє працювати з HTML розміткою для створення структури сторінки веб-додатку.

    2. React - бібліотека JavaScript для створення інтерактивних інтерфейсів (UI) за допомогою компонентного підходу, що дозволяє повторно використовувати код, керувати станом програми і малювати динамічні дані без перезавантаження сторінки.


</details>
